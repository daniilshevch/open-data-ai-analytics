# Система комплексного аналізу макроекономічних прогнозів України (2020-2021)

Мета проєкту — забезпечити 100% точність даних.

## Мета проєкту
Метою цієї роботи є створення автоматизованого конвеєра для обробки та аналізу відкритих даних за допомогою методології **Git Flow**. Проєкт охоплює етапи завантаження, перевірки якості, дослідження та візуалізації економічних показників.

## Джерело даних
У проєкті використовуються дані з Єдиного державного веб-порталу відкритих даних [data.gov.ua](https://data.gov.ua/).
**Набір даних:** Прогнозні макропоказники економічного і соціального розвитку України на 2020-2021 роки (за різними сценаріями).

## Питання та гіпотези для аналізу
Згідно з інструкцією до лабораторної роботи, аналіз зосереджений на наступних гіпотезах:

1. **Порівняння сценаріїв:** Наскільки суттєво відрізняється прогноз номінального ВВП між найбільш оптимістичним (Сценарій 2) та песимістичним (Сценарій 3) прогнозами на 2021 рік?
2. **Аналіз інфляційних очікувань:** Як змінюється індекс споживчих цін (грудень до грудня) залежно від обраного сценарію розвитку економіки?
3. **Соціальний добробут** Порівняння прогнозів заробітної плати між сценаріями.

## Структура репозиторію
* `data/` — сирі та оброблені дані (ігноруються Git).
* `notebooks/` — Jupyter Notebooks для експериментів.
* `src/` — модулі `data_load`, `data_quality_analysis`, `data_research`.
* `reports/figures/` — згенеровані графіки та звіти.

Статус даних: Перевірено на пропуски, дані чисті.

Висновки: проведено комплексний і всебічний аналіз макропоказників України.
